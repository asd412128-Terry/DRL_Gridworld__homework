# DRL_Gridworld__homework

這是一個基於 **強化學習 (Reinforcement Learning)** 中的 **價值迭代 (Value Iteration)** 演算法開發的互動式網頁工具。使用者可以自定義網格地圖，並即時觀察演算法如何計算出每個狀態的價值與最佳行動路徑。

## 🌟 核心功能
* **動態網格生成**：支援 $5 \times 5$ 至 $9 \times 9$ 的自定義尺寸。
* **互動式設定**：透過簡單點擊即可設定「起點」、「終點」及「障礙物」。
* **演算法視覺化**：
    * **價值矩陣 (Value Matrix)**：顯示每個格子收斂後的預期回報數值。
    * **策略矩陣 (Policy Matrix)**：以箭頭標示該位置的最佳移動方向。
* **路徑追蹤**：計算完成後，系統會自動以 **黃色高亮** 標示出從起點到終點的最佳路徑。

## 🧠 演算法原理
本專案核心邏輯參考 `DRL_HW_GridWorld.py` 中的價值迭代演算法：

1.  **狀態初始化**：所有非終點格子的初始價值為 $0$。
2.  **獎勵機制 (Rewards)**：
    * 到達終點：$+10$。
    * 撞擊牆壁或障礙物：$-1$。
    * 每步移動代價：$-0.1$。
3.  **貝爾曼方程 (Bellman Equation)**：
    $$V(s) \leftarrow \max_a \sum_{s', r} p(s', r|s, a) [r + \gamma V(s')]$$
    * 其中折扣因子 $\gamma = 0.9$。
4.  **迭代收斂**：預設執行 $100$ 次迭代以確保數值穩定。

## 🛠️ 技術棧
* **前端**：HTML5, CSS3, JavaScript (負責 UI 渲染與核心演算法運算)。
* **邏輯參考**：Python 3.x, Flask, NumPy。

## 📖 使用步驟
1.  **設定尺寸**：輸入 $n$ 值並點擊「生成網格」。
2.  **標記地圖**：
    * 點擊第 1 下：設定 **起點** (綠色)。
    * 點擊第 2 下：設定 **終點** (紅色)。
    * 後續點擊：設定 **障礙物** (灰色)，數量上限為 $n-2$ 個。
3.  **執行計算**：點擊「計算最佳路徑」，右側將顯示策略，地圖會高亮顯示路徑。
